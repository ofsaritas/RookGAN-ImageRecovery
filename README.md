# RookGAN-ImageRecovery
Dataset and code for the paper:
**"Image Recovery from Severe Cropping Using Rook Polynomials and Pix2PixGAN"**
*Submitted to Information Sciences*
## Abstract
A two-stage hybrid framework for image authentication and recovery under severe cropping attacks, achieving 26.03 dB PSNR at 80% cropping severity.
## Dataset
This repository contains the list of images used in experiments:
- **COCO:** 650 images from [COCO Dataset](https://cocodataset.org)
- **FFHQ:** 650 images from [FFHQ Dataset](https://github.com/NVlabs/ffhq-dataset)
- **NWPU:** 650 images from [NWPU-RESISC45](https://gcheng-nwpu.github.io)
